# Mini-Lend_-Defi-project-
A simple smart contract where users can:  Deposit ETH as collateral, Borrow a mock stablecoin (MiniUSD) up to 50% of their Collateral, Repay loan to unlock collateral, Get liquidated if collateral drops in value up to 70%.
It has the following buttons which maps to solidity smart contract;
	-- Connect 
	-- Stake Collateral
	-- Borrow USD
	-- (Approve) Repay USD
	-- Withdraw
